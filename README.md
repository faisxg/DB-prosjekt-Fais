# DB-prosjekt
## Idéen bak appen:

Ulike lister, der man kan opprette og vise listene man har laget for en enkel person.

Egnet til å være en veldig enkel, kjapp og minimalistisk listeapp.

## Innsamling av persondata
Appen tar opp personlige opplysninger som fornavn, etternavn, e-post og alder, men disse vil benyttes utenfor koden (som f.eks. å veksle personlig info til andre partier), men heller til selve koden eller til kontakt av person.
